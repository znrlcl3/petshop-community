XSS 필터 테스트

즉시 테스트 (클릭만 하면 됩니다):

스크립트 공격 테스트 이미지 이벤트 공격 자바스크립트 URL 안전한 HTML 전체 테스트 필터 상태

📋 테스트 패턴 확인:

  • 스크립트 공격: <script>alert('XSS')</script>
  • 이미지 이벤트: <img src="x" onerror="alert('XSS')">
  • JS URL: <a href="javascript:alert('XSS')">링크</a>
  • 안전한 HTML: <p><strong>안전한</strong> HTML</p>

사용법:

  1. 위의 버튼들을 클릭하면 새 탭에서 테스트 결과를 볼 수 있습니다
  2. 서버 콘솔에서 필터링 로그를 확인하세요
  3. XSS 공격이 차단되는지 확인하세요
기대 결과:
  • 위험한 패턴들 → 차단되거나 무력화됨
  • 안전한 HTML → 그대로 유지됨
  • 필터 상태 → "정상 작동 중" 표시
